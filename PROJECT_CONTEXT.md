## Технический стек

- **HTML/CSS/JS** — без фреймворков и зависимостей
- **CSS**: переменные (`:root`), `clamp()`, grid, flexbox, `aspect-ratio`, `backdrop-filter`, `env(safe-area-inset-bottom)`
- **JS**: IntersectionObserver (scroll reveal), FAQ-аккордеон, бургер-меню, sticky header/bar, трекинг целей Яндекс Метрики
- **SEO**: Schema.org (`ProfessionalService` + `FAQPage`), Open Graph, Twitter Cards, canonical, `robots.txt`, `sitemap.xml`
- **Адаптивность**: breakpoints — 400px, 480px, 768px, 1024px; `prefers-reduced-motion`
- **Деплой**: GitHub Pages (статика)

## Что готово

- Полностью свёрстанный лендинг со всеми 12 секциями
- CSS со всеми адаптивными breakpoint'ами и доступностью (`:focus-visible`)
- JS: аккордеон, бургер-меню, scroll reveal, sticky-бар, трекинг кликов
- Schema.org разметка (LocalBusiness + FAQPage) в JSON-LD
- Фото сжаты и размещены в `site/img/`
- `thanks.html` с `noindex`
- `robots.txt` + `sitemap.xml`

## Что осталось (плейсхолдеры для замены)

| Плейсхолдер | Где встречается | Когда заменить |
|---|---|---|
| `YOUR-DOMAIN.ru` | canonical, og:url, og:image, twitter:image, Schema.org image/url, robots.txt, sitemap.xml | После деплоя на GitHub Pages или подключения домена |
| `YOUR_YANDEX_VERIFICATION_CODE` | `<meta name="yandex-verification">` в index.html | После регистрации в Яндекс Вебмастере |
| `00000000` (ID Метрики) | Закомментированный блок, строки 161–177 index.html | После создания счётчика в Яндекс Метрике |
| Файл `CNAME` | Отсутствует в корне site/ | Только при подключении своего домена |

## Ключевые решения

- Фоновое изображение hero — `video_thumb_02.jpg` (preload в `<head>`)
- WhatsApp-ссылки с предзаполненным URL-encoded сообщением, уникальным для каждой услуги
- Карта — iframe Яндекс Карт с привязкой к OID организации (`oid=117586873400`)
- Favicon — inline SVG с emoji грузовика (zero HTTP requests)
- Яндекс Метрика — в комментарии, раскомментировать после получения ID
- Трекинг кликов WhatsApp/Telegram/телефон — уже прописан в `main.js`, автоматически работает при активации Метрики

## Контекст масштабирования

Проект задуман как **шаблон для тиражирования** на другие сервисные бизнесы. `ИНСТРУКЦИЯ.md` описывает полный цикл: сбор данных с Яндекс Карт → вёрстка → SEO → деплой → аналитика. Для нового проекта достаточно скопировать `site/`, заменить данные клиента и фото.
