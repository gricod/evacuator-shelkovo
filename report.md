## 2. Описание услуг

### Основная деятельность
Эвакуация легковых автомобилей, внедорожников, кроссоверов. Работают круглосуточно без выходных и праздников.

### Подробные описания по направлениям

**Эвакуатор по Щёлково**
Эвакуация заблокированных, повреждённых, не заводящихся автомобилей. Лебёдка, сдвижная платформа. Аккуратная погрузка.

**Эвакуатор Щёлково — Москва**
Эвакуация автомобилей из Щёлково в любой район Москвы. Бережная перевозка. Забирают с любого адреса в Щёлково и районе. Доставляют до нужного автосервиса или адреса в Москве. Работают по Щёлковскому шоссе, Ярославскому шоссе.

**Эвакуатор Фрязино**
Эвакуация автомобилей по Фрязино и из Фрязино в Щёлково, Москву и другие города Подмосковья. Сдвижная платформа, лебёдка для сложных случаев. Подача эвакуатора от 15-20 минут. Перевозят легковые автомобили, внедорожники, мотоциклы.

**Эвакуатор на Щёлковском шоссе**
Эвакуация автомобилей на Щёлковском шоссе в любое время суток. Заберут с обочины, из кювета, после ДТП. Подача от 10-15 минут в зону Щёлково — Балашиха. Сдвижная платформа, лебёдка для сложных случаев. Работают без выходных.

**Эвакуация после ДТП**
Эвакуация заблокированных, повреждённых, не заводящихся автомобилей. Лебёдка, сдвижная платформа.

## 12. Как провести анализ профиля (методология)

### 12.1. Получение данных через API Яндекс Карт

**Шаг 1 — Получение cookies и CSRF-токена:**
```bash
curl -s -c cookies.txt \
  "https://yandex.ru/maps/api/search/1.x/?text=dummy&lang=ru_RU&format=json" \
  -A "Mozilla/5.0 (iPhone; CPU iPhone OS 17_0 like Mac OS X) AppleWebKit/605.1.15"
```
Ответ: `{"csrfToken":"abc123..."}`

**Шаг 2 — Запрос к HTML-странице организации (через yandex.fi):**
```bash
curl -s -L "https://yandex.fi/maps/org/117586873400/" \
  -A "Mozilla/5.0 (iPhone; CPU iPhone OS 17_0 like Mac OS X) AppleWebKit/605.1.15" \
  -o org_page.html
```
Домен `yandex.fi` (международный) не показывает CAPTCHA для одиночных запросов.

**Шаг 3 — Извлечение структурированных данных из HTML:**
```bash
# Рейтинг
grep -o '"ratingValue":[0-9.]*' org_page.html

# Телефон
grep -o '"number":"+7[^"]*"' org_page.html

# Адрес
grep -o '"address":"[^"]*"' org_page.html

# Количество отзывов
grep -o '"reviewCount":[0-9]*' org_page.html

# Цены
grep -o '"price":"[0-9]*","currency":"[^"]*"' org_page.html

# Все уникальные названия полей
grep -o '"name":"[^"]*"' org_page.html | sort -u
```

### 12.2. Скачивание всех фотографий

**Шаг 1 — Извлечение URL фотографий:**

### 12.3. Получение всех отзывов (динамическая подгрузка)

Отзывы сверх первых 3-5 подгружаются через AJAX. Для полного сбора нужен браузерный движок:

```python
# pip install playwright
from playwright.sync_api import sync_playwright

with sync_playwright() as p:
    browser = p.chromium.launch(headless=True)
    page = browser.new_page()
    page.goto("https://yandex.ru/maps/org/117586873400/reviews/")

    # Прокрутка для подгрузки всех отзывов
    for _ in range(20):
        page.mouse.wheel(0, 3000)
        page.wait_for_timeout(1000)

    # Извлечение текста отзывов
    reviews = page.query_selector_all(".business-review-view__body-text")
    for r in reviews:
        print(r.inner_text())

    browser.close()
```

## 13. Как скопировать информацию (экспорт)

### 13.1. Быстрое копирование через CLI

**Экспорт контактов в JSON:**
```bash
PYTHONIOENCODING=utf-8 python -c "
import re, json
with open('org_page.html', 'r', encoding='utf-8') as f:
    html = f.read()

data = {
    'name': 'Эвакуация автомобилей',
    'phone': re.findall(r'\"number\":\"([^\"]+)\"', html),
    'address': re.findall(r'\"address\":\"([^\"]+)\"', html),
    'rating': re.findall(r'\"ratingValue\":([0-9.]+)', html),
    'reviews_count': re.findall(r'\"reviewCount\":(\d+)', html),
    'prices': [],
}

for m in re.finditer(r'\"price\":\"(\d+)\",\"currency\":\"([^\"]+)\"', html):
    ctx = html[max(0,m.start()-500):m.start()]
    titles = re.findall(r'\"title\":\"([^\"]+)\"', ctx)
    data['prices'].append({
        'service': titles[-1] if titles else 'N/A',
        'price_from': int(m.group(1)),
        'currency': m.group(2)
    })

print(json.dumps(data, ensure_ascii=False, indent=2))
" > org_data.json
```
